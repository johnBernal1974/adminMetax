¿Por qué elegir Metax?

Metax nace con el objetivo de mejorar la experiencia del transporte en taxi, combinando tecnología, transparencia y seguridad tanto para usuarios como para conductores.

  • Plataforma diseñada específicamente para transporte en taxi.
  • Tarifas definidas y visibles, sin sorpresas.
  • Seguimiento del viaje en tiempo real para mayor tranquilidad.
  • Uso responsable de los datos y respeto por la privacidad.
  • Soporte y canales oficiales claramente definidos.

Para usuarios

Solicita tu taxi de forma sencilla, conoce la tarifa, visualiza el recorrido y viaja con mayor confianza.

Para conductores

Recibe solicitudes cercanas, mantente visible durante el servicio y mejora tus oportunidades de ingreso usando una plataforma confiable.

Seguridad y confianza

Implementamos reglas de uso, reportes y medidas de seguridad para proteger a todos los actores del servicio.
