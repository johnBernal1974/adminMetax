Política de privacidad

Última actualización: 29/01/2026

Esta política explica cómo Metax recopila y utiliza información cuando usas nuestros servicios.

1. Datos que podemos recopilar

  • Datos de cuenta: teléfono, identificadores de usuario, y datos mínimos para autenticación.
  • Datos de uso: eventos técnicos para mejorar estabilidad (por ejemplo, errores).
  • Datos operativos necesarios para la prestación del servicio (según el caso de uso).

2. Finalidades

  • Prestar el servicio y permitir el acceso a funcionalidades.
  • Seguridad, prevención de fraude y soporte.
  • Mejoras del producto (analítica técnica y rendimiento).

3. Compartición

Solo compartimos datos cuando es necesario para operar el servicio (proveedores tecnológicos) o por obligación legal.

4. Retención

Conservamos la información el tiempo necesario para prestar el servicio y cumplir obligaciones legales o de seguridad.

5. Derechos del titular

Puedes solicitar acceso, corrección o eliminación de tu información. Para eliminación de cuenta, visita Eliminar cuenta / Darse de baja.

6. Contacto

Para preguntas o solicitudes: revisa Soporte.
